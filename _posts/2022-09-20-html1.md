## ```HTML``` (```Hyper Text Transfer Protocol```) : 웹 문서의 모양을 기술하기 위해 사용하는 마크업 언어
- 인터넷에서 데이터를 주고받을 수 있는 텍스트 기반의 통신 프로토콜 (```Transfer Protocol```)
- 제목, 단락, 목록 등과 같이 본문을 위한 구조적 의미를 표현할 수 있고, 링크, 인용 등을 통해 구조적 문서를 구성할 수 있음

입력 : 

```
<html>
	<head>
		<div style="
			color: blue;
			font-size: 15px;
		">
			Hello World!
		</div>
	</head>
</html>
```

출력 :

<html>
	<head>
		<div style="
			color: blue;
			font-size: 15px;
		">
			Hello World!
		</div>
	</head>
</html>

- ### HTML 엘리먼트 (```HTML Elements```) : HTML 문서나 웹 페이지를 이루는 개별적인 요소
- ### HTML 태그 (```HTML Tags```) ```<tag></tag>``` : 엘리먼트의 시작과 끝을 표시하는 마크업 기호
- ### HTML 속성 (```HTML Attributes```) ```<tag element="">``` : 여는 태그 내에서 명령어를 구체화하는 속성
- ### HTML 변수 (```HTML Arguments```) ```<tag element="value">``` : 태그 내의 속성에 해당되는 변수
- ### HTML 주석 ```<!-- 주석 -->```

## ```<html>``` : HTML으로 작성된 문서임을 선언하는 태그

```
<html>
...
</html>
```

### ```<!DOCTYPE html>``` : 마크업 언어에서의 문서 형식을 정의하는 태그
<hr>

- DTD 태그 (Document Type Definition)
- 웹 브라우저의 렌더링 엔진이 관용 모드 (Quirks Mode)가 아닌 표준 모드로 렌더링
- HTML5 :```<!DOCTYPE html>```으로 선언

```
<!DOCTYPE html>
...
</html>
```

### ```<html lang="">``` : 웹 문서의 텍스트가 어떤 언어로 작성되었는지 나타내는 속성
<hr>

```
<!DOCTYPE html lang="ko">
...
</html>
```

## ```<head>``` : 웹 문서의 속성을 선언하는 태그
- 실제 웹 브라우저 화면 상에는 출력되지 않음
- HTML 문서의 부가적인 정보들을 넣음

```
<!DOCTYPE html lang="ko">
	<head>
	...
	</head>
</html>
```

### ```<title>``` : 웹 문서의 제목을 나타내는 태그
<hr>

- 브라우저의 탭에 표시되는 제목을 출력
 
```
<title>
	웹 문서 제목
</title>
```

#### ```<meta name="">``` : 웹 서버가 웹 브라우저의 요청에 대해 응답할 정보들을 선언하는 속성
<hr>

```

	name="..."

/>
```

##### ```<meta name="description">``` : 웹 브라우저의 검색 결과에 표시될 웹 문서의 설명을 선언하는 name 변수
<hr>

- content 값은 공백을 포함해 최대 150글자여야 함

```
<meta
	name="description"
      	content="설명"
/>
```

<hr>

```
<meta
    name="keywords"
      	content="키워드1, 키워드2,..."
/>
```

