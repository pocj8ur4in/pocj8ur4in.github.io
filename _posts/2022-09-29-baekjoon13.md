## 1. 문제 조건
아래 예제와 같이 고양이를 출력하시오.

- 입력 : 없음.
- 출력 :
  
```
\    /\
 )  ( ')
(  /  )
 \(__)|
```

## 2. 문제 풀이

```
print("\    /\\")
print(" )  ( ')")
print("(  /  )")
print(" \(__)|")
```
