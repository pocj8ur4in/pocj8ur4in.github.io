## 1. 문제 풀이

```
#include <iostream>
using namespace std;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);
    cout.tie(NULL);
    
    int A, B;
    
    cin >> A >> B;
    
    cout << A - B;
    
    return 0;
}
```

- ```<iostream>``` 클래스의 ```cin```와 ```cout```로 입출력 구현
