### 프로젝트 소개

**pocj8ur4in.github.io**는 [TypeScript](https://www.typescriptlang.org/)와 [React](https://reactjs.org/)를 기반으로 하는 블로그를 개발하는 프로젝트입니다. 이 프로젝트는 기존에 운영하던 [Jekyll](https://jekyllrb-ko.github.io) 기반 블로그의 성능과 유지보수성을 개선하기 위해 시작되었습니다. React와 TypeScript를 도입하여 더 빠르고 직관적인 개발 환경을 제공하고, 확장성과 재사용성을 높여 블로그의 기능을 향상시키는 것을 목표로 하고 있습니다.
